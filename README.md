# Project Title: Stellar Movie DApp

## Project Description

Stellar Movie DApp is a decentralized application (DApp) based on a smart contract built on the Stellar web3 blockchain using the Soroban SDK. This platform provides a secure and persistent system for adding movie data (including poster images via the UI), and provides a space for users to upload reviews (comments) and ratings for each movie transparently directly on the blockchain network.

## Project Vision

The vision of this project is to decentralize the movie database system and review community in the digital era, ensuring transparency and preventing rating manipulation by centralized parties. By leveraging the efficiency of the Stellar network, every review and data entry is permanent (immutable), thereby creating an authentic and trustless movie rating platform.

## Project Features List

- **Create Movie:** A feature to add a new movie using a Smart Contract, storing the Title, Release Year, Description, and a Base64 Poster Image.
- **Get Movie Details:** An efficient search system utilizing a unique ID to fetch complete movie details in real-time from blockchain storage.
- **Add & View Comments (Rating System):** A facility for audiences to add text comments and numerical scores (1-5), as well as load the review history from other users.
- **Web3 Wallet Integration:** Built with decentralized network authentication readiness (such as Freighter Wallet) to process transaction signing.
- **Modern Glassmorphism UI:** A premium, bright, interactive, and mobile-friendly front-end design.
